FIDIMAG

Finite difference atomistic and micromagnetic solver

A powerful, open-source simulation package

Fidimag is free to use under the BSD license, scriptable in Python, and ready to use.

Easy to script


Fidimag is provided with an intutive Python interface. Behind the scenes, C and OpenMP do the heavy lifting. This means that Fidimag is fast, efficent, and easy to script.

Atomistic or micromagnetic?

Define your magnetic system and choose from finite-temperature, atomistic simulations or continuous micromagnetic solvers. Switching between the two is easy and the source code is clearly seperated. So it is easy to dive into the details.

Finite difference


The simulation is based on the finite difference method, and meshes can be defined on cubic and hexagonal lattices. Fidimag can also handle 3D geometries.

Documentation


We provide extensive documentation and comprehensive examples to help you get up and running as quickly as possible.
